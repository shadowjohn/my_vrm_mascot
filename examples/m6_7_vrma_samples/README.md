# M6.7 VRMA Samples

This folder documents local-only demo samples for `motion_template_lab.html`.
Third-party VRMA binaries are not committed to git. Keep restored `.vrma`
files under `local_assets/vrma/`, or temporarily copy them here for legacy
manual checks.

Source:

```text
https://github.com/tk256ailab/vrm-viewer/tree/main/VRMA
```
