使用方法 / Skill

本機開發以 `server.py` 為主；正式 `index.html` 是 Workbench 入口，方便新人或未來部署時快速找到工具。

本機啟動

cd /d D:\mytools\my_vrm_mascot
run_server.bat
open_portal.bat
stop_server.bat

手動啟動

cd /d D:\mytools\my_vrm_mascot
python .\server.py

http://127.0.0.1:8765/

Server / 3wa 搬移邊界

先整理成 local-first Workbench，不急著改後端。搬進 3wa 時分清楚靜態展示與寫入能力。

靜態展示可直接搬移

HTML、JS、VRM、VRMA、JSON、PNG 等靜態資源可放到 Apache2 / 3wa 目錄，由瀏覽器直接讀取。

寫入需要 API

寫入 mining_log.json、產生 candidates/*.json 或上傳 VRMA 時，需要 PHP API 或 Python API；否則 browser 只能下載檔案。
